Snap! Development

Snap! is a visual programming language that was developed at UC Berkeley. More information about Snap! can be found here.

In my first semester at Cal, I took CS10: The Beauty and Joy of Computing, which is a class based around Snap!. Because Snap! was my first introduction to programming, it has since held a special place in my heart. Now in my third semester, I am taking a student-run class called Snap! Development. In this class, I have had an introduction to Javascript and learned how to sort through an enormous corpus of code.

Our first small project involved adding a block to Snap!. I chose to add a square root block, screenshotted above. In coming weeks, I will also be working to add an Undo function to the program.
